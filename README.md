# Hello World App

A minimal Vite + React + TypeScript starter app.

## Project Overview

This repository contains a small React application scaffolded with Vite and TypeScript. It's intended as a simple starter for development and learning.

## Prerequisites

- Node.js (16+ recommended)
- npm or yarn

## Setup

Install dependencies:

```
npm install
```

## Development

Start the dev server with hot reload:

```
npm run dev
```

Open http://localhost:5173 (or the port shown in the terminal).

## Build

Create a production build:

```
npm run build
```

Preview the production build locally:

```
npm run preview
```

## Project Structure

- `index.html` — App entry HTML
- `src/` — Source files
  - `main.tsx` — App bootstrap
  - `App.tsx` — Root component
  - `index.css`, `App.css` — Styles

## Scripts

Common scripts (from `package.json`):

- `dev` — Start Vite dev server
- `build` — Build for production
- `preview` — Preview production build
